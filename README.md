# NBA Oracle

ML-powered NBA game predictions.

## Tech Stack

- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: FastAPI + Python 3.13
- **ML**: scikit-learn + Jupyter notebooks
- **Infrastructure**: Docker + 1Password

## Quick Start

```bash
# Install task runner (if not installed)
brew install go-task

# First time setup
task setup

# Start development environment
task dev
```

## Features

- Real-time NBA game predictions
- Machine learning models trained on historical data
- Clean, responsive UI
- RESTful API

## Development

| Command | Description |
| ------- | ----------- |
| `task dev` | Start development environment |
| `task clean` | Stop and clean up |
| `task env` | Sync environment variables from 1Password |
| `task logs` | View container logs |
